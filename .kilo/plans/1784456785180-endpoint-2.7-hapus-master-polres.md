## Step 2 — Controller Method (`Master.php`)

Add `polres_delete($polres_id)` after `polres_put()`.

### Sub-steps (in order):

1. **JWT payload extraction**: `$payload = get_jwt_payload($this);` — same pattern as `polres_post()`.
2. **Role gate**: if `$payload === null || !isset($payload['role_id']) || $payload['role_id'] !== 1` → HTTP 403 + `{"status":403,"message":"Akses ditolak. Anda tidak memiliki otoritas Super Admin.","data":{}}`.
3. **Disable DB debug**: `$this->db->db_debug = FALSE;` — prevents CI3 crash on SQL error.
4. **Execute DELETE**: `$this->db->delete('tbl_polres', ['polres_id' => $polres_id]);`
5. **Inspect error**: `$error = $this->db->error();`
   - Code `1451` (FK constraint violation) → HTTP 409 + `{"status":409,"message":"Polres tidak dapat dihapus karena masih menaungi personel aktif (Restricted by System).","data":{}}`
   - Code `0` (success) → HTTP 200 + `{"status":200,"message":"Data polres berhasil dihapus.","data":{}}`
6. **Re-enable debug**: `$this->db->db_debug = TRUE;`
7. **Return** after each branch (no fall-through).

### Edge cases / risks:
- `$polres_id` not found → still succeeds (affected_rows = 0). Acceptable — MySQL `DELETE` on non-existent row is not an error. Message remains the same.
- Multiple FK violations possible (not just `tbl_personil`). Code 1451 catches any FK. The error message is generic enough to cover all cases. `ponytail:` If a more specific message per child table is needed later, inspect `$error['message']` for table name.

## Step 3 — Playwright Tests (`master_polres.spec.ts`)

Add 2 new tests after Test 5 (line 146), inside the same `test.describe.serial` block.

### Prerequisite check (inserted before both tests or as first line of Test 6):

Ensure `tbl_personil` FK on `polres_id` exists. If the FK does not exist, the 409 test cannot fire. Two options:
- **Option A (preferred)**: Add to `Seed migration` (`tests/seed.php`) an `ALTER TABLE tbl_personil ADD CONSTRAINT fk_personil_polres FOREIGN KEY (polres_id) REFERENCES tbl_polres(polres_id) ON DELETE RESTRICT;` (run idempotently at seed time).
- **Option B (if FK already exists from prior migration)**: Skip seed change; just use `execSync` raw SQL in tests.

**Decision**: Use `execSync` in each test to run SQL directly against the DB via `mysql` CLI or PHP via PDO. This avoids modifying seed.php and keeps FK dependency explicit in test code.

### Test 6 — `DELETE /api/v1/master/polres/:id — Conflict Trap (409)`

```ts
test('DELETE /api/v1/master/polres/:id — Conflict Trap (409)', async ({ request }) => {
  // 1. Insert dummy personil referencing createdPolresId to trigger FK 1451
  execSync(
    `php -r '
      $pdo = new PDO("mysql:host=127.0.0.1;dbname=sindomondb;charset=utf8mb4", "root", "root");
      // Ensure FK exists (fail silently if already exists)
      try {
        $pdo->exec("ALTER TABLE tbl_personil ADD CONSTRAINT fk_personil_polres FOREIGN KEY (polres_id) REFERENCES tbl_polres(polres_id) ON DELETE RESTRICT");
      } catch (PDOException $e) {
        if (strpos($e->getMessage(), "Duplicate") === false && strpos($e->getMessage(), "already exists") === false) throw $e;
      }
      $uuid = "00000000-0000-0000-0000-" . str_pad(dechex(time()), 12, "0", STR_PAD_LEFT);
      $pdo->exec("INSERT IGNORE INTO tbl_personil (personil_id, nrp, nama_lengkap, pangkat_id, jabatan_id, status_aktif, polda_id, polres_id) VALUES (\"" . $uuid . "\", \'88TRAP99\', \'TRAP_PERSONIL_POLRES\', 1, 1, \'Aktif\', 1, ' . createdPolresId . ')");
    '`,
    { cwd: process.cwd(), timeout: 10000 }
  );

  // 2. Attempt delete as Super Admin
  const start = Date.now();
  const res = await request.delete(`/api/v1/master/polres/${createdPolresId}`, {
    headers: { Authorization: `Bearer ${adminJwt}` },
  });
  assertLatency(start, 'DELETE /master/polres (conflict 409)');

  // 3. Assert
  expect(res.status()).toBe(409);
  const body = await res.json();
  assertEnvelope(body, 'DELETE /master/polres (conflict 409)');
  expect(body.status).toBe(409);
  expect(body.message).toBe('Polres tidak dapat dihapus karena masih menaungi personel aktif (Restricted by System).');
  expect(body.data).toEqual({});
});
```

**Edge case**: If `tbl_personil` table or columns don't exist, the raw PHP insert will throw. Graceful failure is acceptable — the test runner reports the error clearly.

### Test 7 — `DELETE /api/v1/master/polres/:id — Success (200)`

```ts
test('DELETE /api/v1/master/polres/:id — Success (200)', async ({ request }) => {
  // 1. Release FK constraint: delete the dummy personil row
  execSync(
    `php -r '
      $pdo = new PDO("mysql:host=127.0.0.1;dbname=sindomondb;charset=utf8mb4", "root", "root");
      $pdo->exec("DELETE FROM tbl_personil WHERE nrp = \'88TRAP99\'");
    '`,
    { cwd: process.cwd(), timeout: 10000 }
  );

  // 2. Delete polres
  const start = Date.now();

```

**Edge case**: If previous test failed and trap personil was never inserted, this DELETE still runs on the orphaned polres row — success path still works.
